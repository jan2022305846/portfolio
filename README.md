# Portfolio

A modern, responsive portfolio website showcasing my skills in Networking, System Administration, Cybersecurity, and Development. Built with React and Vite for optimal performance.

## Features

- **Responsive Design**: Optimized for all devices with a clean, professional layout.
- **Dark Mode Toggle**: Seamless theme switching for better user experience.
- **Interactive Sections**: Hero, About, Skills, Projects, and Contact sections with smooth animations.
- **Star Background**: Dynamic animated star field for visual appeal.
- **Mobile Navigation**: Collapsible menu for mobile devices.

## Tech Stack

- **Frontend**: React 19, Vite
- **Styling**: Tailwind CSS v4
- **Icons**: Lucide React
- **Utilities**: Class Variance Authority, clsx, Tailwind Merge
- **Build Tool**: Vite

## Getting Started

### Prerequisites

- Node.js (version 18 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/portfolio.git
   cd portfolio
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:5173](http://localhost:5173) in your browser.

### Build for Production

```bash
npm run build
```

## Usage

- Navigate through the sections using the navigation bar.
- Toggle between light and dark modes using the theme button.
- View projects and contact information in their respective sections.
